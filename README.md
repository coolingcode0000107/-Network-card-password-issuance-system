# 卡密销售管理系统

一个基于Python + PyQt5 + MySQL + Socket的分布式卡密销售管理系统

## 功能特点

- 用户管理（管理员/普通用户）
- 商品管理（添加/修改/删除）
- 卡密管理（自动生成/激活）
- 订单管理（购买/退款）
- VIP等级系统
- 余额充值系统
- 签到激励
- 收益统计

## 系统要求

- Python 3.7+
- MySQL 5.7+

## 使用说明

1. 管理员功能：
   - 商品管理
   - 用户管理
   - 订单管理
   - 退款处理
   - 收益统计

2. 用户功能：
   - 购买商品
   - 激活卡密
   - 申请退款
   - 余额充值
   - 查看订单
